# 项目文件介绍


## middleware 文件夹

存放所有的中间件

中间件介绍：

- static-files.js -- 处理静态文件的 middleware

- controller.js -- 处理路由文件的 middleware

- rest.js -- 为 ctx  绑定 `.rest()`方法（处理 REST API）的 middleware                                          


## static 文件夹 

存放前端项目


## 运行项目

1. 安装依赖

```
	npm install
```

2. 启动项目

```
	npm start
```
